# Assembly - Traffic Light `DT21-TL`

```{needget}
*   Traffic light components ([Duckietown project shop](https://get.duckietown.com/products/duckietown-traffic-light-dt22-tl))

*   An appropriately [configured SD-card](book-opmanual-duckiebot:setup-duckiebot-sd-card).

*   Tools: (strong) wood glue or hot glue gun, tape, double-sided tape.
---
*   An assembled traffic light in configuration `DT21-TL` or previous legacy versions.
```

(subsec:howto-bom-TL21)=
## Part 1: Preliminaries and Part Overview

Each picture in the assembly guide includes a small yellow box in the top-left corner. This box indicates the parts required for that step.

:::{note}
If a glue icon appears in the yellow box, it means some parts will be permanently connected, regardless of whether you're building a Watchtower (WT) or a Traffic Light (TL). While actual glue might not be necessary, we recommend a rigid connection for stability.
:::
